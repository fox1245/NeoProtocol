NeoProtocol Federated Mode — PoC

Two browsers run agents and talk to each other over a P2P WebRTC data channel using the ACP wire format (Zed Agent Client Protocol — JSON-RPC 2.0, NDJSON-style). Either signal through the Originator (Standard Mode) or via a copy-pasted URL (Minimal Mode, zero runtime server). See SPEC §16 and docs/federated-mode.md for the full design + cross-network safety profile.

Standard Mode — Originator signaling
