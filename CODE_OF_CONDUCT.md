# Code of Conduct

This project follows the [Contributor Covenant v2.1][cc] as its code
of conduct. The full text is at the link below; the short version:

- Be welcoming and respectful in all project spaces (issues, PRs,
  discussions, code review comments).
- Critique ideas, not people. Disagreement is fine; personal attacks
  are not.
- Assume good faith — most misunderstandings are honest.
- If you see behavior that violates this, email the maintainer (see
  the GitHub profile of the repository owner). Reports are kept
  confidential.

Maintainers may remove comments, commits, code, issues, or block
contributors whose behavior is judged inconsistent with this code.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
